## Environment
- OS: MacOS El Capitan
- Platform: Python 3
- Librarys: 
	- OpenCV 3
	- appscript
	- pyautogui

## How to run it?
- run it in python
- press `'b'` to capture the background model (Remember to move your hand out of the blue rectangle)
- press `'r'` to reset the backgroud model
- press `'ESC'` to exit
